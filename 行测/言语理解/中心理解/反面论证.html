反面论证 ​

字数: 5400 字 时长: 20 分钟 VIP文档

一、什么是反面论证 ​

  1. 反面论证是藏文修辞学上论证修饰法的一个分类。就是从论点的对立面入手证明相反论点的不成立，从而证明正面观点的正确性。作者常常会为了证明自己论点的可靠性用到这种方法，使自己的论点更加突出。

二、常见标志词和类型 ​

  1. 1、标志词：
    1. （1）否则；不然；一旦
    2. （2）如果(不、无、没有)……，那么/就……（后接负面结果）；倘若……，可能/便…… 如果不处理环境污染，那么我们的后代将失去生存的家园。
    3. （3）没有……就没有……；不……则不……
  2. 2、类型：
    1. （1）一般类格式：正面观点/对策 +如果/倘若/一旦 + 不良结果
    2. （2）特殊类格式：如果不/没有/缺乏……，就/将会/将使……

三、解题技巧 ​

  1. 1、有观点类（一般类格式）：观点/对策 + 反面论证。反面论证用来论证观点的，反面论证不重要，观点/对策重要。
  2. 2、无观点类（特殊类格式）：背景陈述/客观问题 + 反面论证。反面论证中的对策重要，把“错误的做法”，反过来，即为对策。
    1. 注意：反面论证的结果是消极的，围绕这个负面结果设置的选项不要选

四、随笔练习 ​

例1：（2016江苏）内容产业包括三个层次：信息、知识和智慧。报业更多是生产信息，出版业主要集中生产知识和智慧，生产信息的必然先于生产知识和智慧的受到互联网冲击。其实原先不少报纸有在知识层面生产的传统，但近年来，报纸的这一特色日益褪色。如果报纸不能从知识层面生产信息，只是在海量信息层面做文章，竞争不过新媒体是自然而然的事。

  1. 这段文字重在说明：
  2. A. 报业竞争不过新媒体是毋庸置疑的事情
  3. B. 报业相比出版业更早受到互联网的冲击
  4. C. 互联网的海量信息是其取胜的关键武器
  5. D. 今天报业欲生存应多在知识层面生产信息
  1. 文段首先交代背景，说明内容产业包括的三个层次，随后通过转折关联词“其实”指出“报纸在知识层面生产不足”这一问题。
  2. 文段尾句通过反面论证提出对策，即“报纸应该从知识层面生产信息”，对应D项。
  3. A项：“报业竞争不过新媒体”是反证部分的后果，属于假设变现实，排除。
  4. B项：为转折前的内容，非重点，排除。
  5. C项：“海量信息”与文段核心话题不符，文段主题词为“报业”，排除。
  6. 故正确答案为D。

例2：（2017新疆）近年来，我国推行了一系列改革，为双创营造制度环境。大学生、海归、大企业高管和连续创业者、科技人员这支”新四军”的崛起，可以看作是创业主体从精英走向大众的一个强有力的信号。不过，因为机制体制不健全，一些法律、法规、政策存在矛盾，人的价值的最终实现还存在着不少障碍。倘若人的价值迟迟不能得到充分保障，创新创业者才能的释放势必会受到影响，诸多科技成果就可能烂在抽屉里，诸多科技企业就可能因为缺乏创新而死去。

  1. 这段文字意在强调（）
  2. A. 改革要为双创营造制度环境
  3. B. 创业主体将从精英走向大众
  4. C. 机制体制阻碍人的价值实现
  5. D. 保障人的价值是双创的前提
  1. 文段首先交代背景，指出我国通过改革为“双创”营造制度环境及创业主体发生变化，随后通过“不过”进行转折，提出“人的价值的最终实现还存在不少障碍”的问题。
  2. 尾句通过反面提出对策，强调要充分保障人的价值，故文段重点强调保障人的价值对于双创的必要性，对应D项。
  3. A项为转折前的表述，且“要”时态错误，根据文段首句可知，我国已经为双创营造了制度环境，排除；
  4. B项为转折前的表述，且没有提到主题词“人的价值”，排除；
  5. C项为问题的表述，非重点，且没有提到“双创”这一话题，排除。
  6. 故正确答案为D。

例3：（2019广东选调）当前，新闻媒体正在深入推进融合发展。新闻作品版权是媒体的核心资产，但目前加剧的侵权现实让人无奈。作为新闻媒体，一般乐见自己的作品被广泛传播，但前提是尊重原创劳动，并得到授权方能使用。如果对未经授权擅自使用、传播新闻作品的行为置之不理，传统产业的发展将受到严重影响，媒体的融合发展也会面临严重困难。

  1. 这段文字意在强调（ ）。
  2. A.新闻媒体的融合有利于版权保护
  3. B.新闻媒体的融合发展面临严重困难
  4. C.新闻媒体应避免自己的作品被随意转载
  5. D.推动新闻媒体融合发展，必须做好版权保护
  1. 文段首先引出媒体融合的话题，接着指出现在新闻作品侵权的问题。
  2. 后文通过 “如果对未经授权擅自使用、传播新闻作品的行为置之不理，传统产业的发展将受到严重影响，媒体的融合发展也会面临严重困难”进行反面论证提对策，即强调的是我们要做好版权的保护，对应答案D项。
  3. A项，文段开头指出新闻媒体融合带来侵权加剧的现象，表述与文意相悖，排除。
  4. B项，文段末尾说的是如果不做好版权保护会使媒体融合发展有困难，而不是说一定有困难，假设变现实，排除。
  5. C项，文段提到“作为新闻媒体，一般乐见自己的作品被广泛传播”，表述不符合文意，排除。
  6. 故正确答案为D。

例4：（2017山东）无论出于什么原因，网络语言低俗化都对网络文明建设造成了危害，甚至会从网上转到网下，降低整个社会的文明程度。当前，网络语言的发展路径已经很清晰：从虚拟空间进入口语表达，再进入书面语，最终有可能沉淀到语言应用的各个方面。如果任由网络低俗语言发展，久而久之它们就会成为约定俗成的惯用语，其负面效应不容忽视。

  1. 这段文字意在说明：
  2. A.网络语言低俗化的负面效应已开始凸显
  3. B.网络语言低俗化将会影响社会文明程度
  4. C.应该警惕网络语言向习惯语转化的可能
  5. D.急需采取措施治理网络语言低俗化趋势
  1. 文段开篇指出网络语言低俗化会对社会造成危害，降低文明程度，后文“当前······沉淀到语言应用的各个方面”解释说明为何网络语言低俗化会对社会造成危害，尾句从反面提出对策，即不能任由网络语言低俗化发展，要对其进行管理，对应D项。
  2. A项：“网络语言低俗化的负面效应”为问题表述，非重点，排除；
  3. B项：“将会影响”为问题表述，且对应文段首句，非重点，排除；
  4. C项：文段讨论的核心话题为“网络语言低俗化”，而非“网络语言”，该项偏离文段核心话题，且概念扩大，排除。
  5. 故正确答案为D。

例5：（2018黑龙江）虽然说经营性养老机构的定价是放开的，政府不能够干预，但是，从保障购买者权益、稳定养老床位价格、规范市场秩序等角度来说，有关方面需要高度警惕这种销售床位的经营模式带来的种种问题。比方说，床位可以炒卖，既有可能背离了养老机构床位的属性——把养老服务变成一种投资形式，还有可能把养老机构床位的价格哄抬高，造成老人们买不起也住不起。另外，床位售价被炒高后很有可能会出现闲置浪费。总之，如果不加以规范，有可能重蹈中国楼市的炒房覆辙。

  1. 这段文字意在说明（ ）。
  2. A.养老机构炒卖床位将带来各种问题
  3. B.政府应当关注养老机构的床位定价
  4. C.政府应当规范养老机构的经营模式
  5. D.养老机构炒卖床位可能是变相炒房
  1. 文段开篇引出“经营性养老机构定价放开”的话题，随后通过转折词“但是”强调政府“需要高度警惕销售床位的经营模式带来的种种问题”，而后通过例子具体阐述前文问题，文段尾句通过结论词“总之”对前文举例进行总结，通过反面论证提出对策，即需要规范养老机构销售床位的经营模式。故文段重在强调政府需要对养老机构的经营模式加以规范，对应C项。
  2. A项，“炒卖床位将带来各种问题”为问题表述，非重点，排除。
  3. B项，“床位定价”对应举例内容，非重点，且文段主题词为“养老机构的经营模式”偏离核心话题，排除。
  4. D项，“养老机构炒卖床位可能是变相炒房”对应尾句反面论证的不良后果，非重点，排除。
  5. 故正确答案为C。

例6：（2018天津宁河区教师）某公司的经验充分显示出，成功的行销运作除了有赖专门的行销部门外，还需要有优异的产品，精密的市场调研，更少不了专业的业务部门、公关部门、擅长分析的财务部门以及物流后勤等部门的全力配合与支持。如果行销部门独强而其他部门弱，或是行销部门与其他部门不合，或是公司各部门无法有效地整合，都会让行销运作无法顺利有效地进行，难以发挥应有的强大威力。

  1. 这段文字主要强调的是（ ）。
  2. A.行销部门只有与其他部门紧密配合才能更好地发挥作用
  3. B.注重团队合作是该公司取得成功的宝贵经验
  4. C.成功的行销运作可以给企业带来巨大的经济效益
  5. D.该公司各个部门的有效整合是其成功的关键
  1. 文段开篇通过“需要”直接点明对策，并通过程度词“更少不了”强调行销需要多部门的全力配合与支持才能发挥强大的威力，随后通过反面论证再次强调行销不能部门独强，应与各部门配合。反面论证不重要，故文段重点在于强调行销需要多部门紧密配合才能发挥强大的威力，对应A项。
  2. B项，没有提到文段的主题词“行销”，与文段核心话题不一致，排除；
  3. C项，文段未提及“行销给企业带来的经济效益”，无中生有，排除；
  4. D项，没有提到文段的主题词“行销”，与文段核心话题不一致，排除。
  5. 故正确答案为A。

例7：（2019青海法院检察院22%）按理说，商家挽留客户，就要拿出足够的诚意，努力以优质、廉价的服务吸引客户“用脚投票”。但一些APP平台偏偏反其道而行之，千方百计通过玩套路、设陷阱等不光彩的手段挽留用户：一不小心就成了会员，被扣款却毫不知情，付费容易、取消时却晕头转向······这不是留人，而是坑人。这种“套路”也注定不得人心，行之不远。

  1. 这段文字的中心主旨是：
  2. A.APP套路越深越留不住用户
  3. B.APP套路形式多多，需谨慎
  4. C.APP吸引用户的方式有待改善
  5. D.APP套路用户亟待整治
  1. 第一句话说明在常理中商家挽留客户应该靠诚意和商品品质。
  2. 第二句话出现了转折关联词“但”，重点来到了转折之后。转折后说明部分APP平台反其道而行之玩套路的这种坑人的手段。
  3. 第三句话总结前文APP平台的做法是在坑人。
  4. 最后一句话虽然没有明显的反面论证的标志词，但是从内容上来看都有“不……，不……”这样否定的描述，并且从最终走不远的结果来看，也是一个消极的不好的结果。
  5. 那么整个文段并没有正面观点，最后一句反面论证就显得至关重要。通过反推能得到：不能有这种“套路”。反推出来的本质是对策。只有D项“亟待”能体现出对策，是文段主旨句的同义替换。
  6. A项可以对应文段最后部分总结的问题，但不属于对策选项，不是最优选，排除。
  7. B项“套路形式多多，需谨慎”为针对用户的对策，文段强调的是商家，排除。
  8. C项“吸引用户的方式有待改善”属于对策选项，但表述不明确，需要针对“APP套路客户”提对策，排除。
  9. 故正确答案为D。

例8：（2021江苏）在全球化时代，面对过剩经济，内部市场是参与全球竞争的后盾。原来中国在全球分工大循环中，主要参与的是生产环节，微笑曲线的两端——上游的资本和下游的市场——都要依赖发达国家。从贸易战可以发现，拥有市场的一方才能拥有全球化竞争的主场，没有市场就会处处受制于人，中国要想获得全球竞争的话语权，就必须有强大的本地需求。

  1. 这段文字意在强调：
  2. A.中国经济发展不应仅仅依赖生产环节优势
  3. B.新的经济形势下中国应更加重视内部市场
  4. C.内部市场占有率是全球经济竞争的新热点
  5. D.中国经济要在全球分工中取得更大主动权
  1. 文段首句引出话题，指出“内部市场”是参与全球竞争的后盾，然后指出原来中国在全球分工中主要参与生产环节，资本和市场要依赖发达国家这个问题。
  2. 接着通过正反两个方面论证，强调拥有市场的重要性。最后尾句通过“必须”提出对策，强调中国要重视“强大的本地需求”即“内部市场”，对应 B 项。
  3. A项，未提及文段论述的核心话题“内部市场”，且“依赖生产环节优势”文段并未提及，排除；
  4. C项，未提及文段论述的核心话题“中国”，且“新热点”文段并未提及，无中生有，排除；
  5. D项，“取得更大主动权”表述不够明确，排除。
  6. 故正确答案为B。

例9：（2016山东）中国正掀起3D打印投资热。面对3D打印的广阔前景，各地纷纷兴建3D打印产业园，并出台各项优惠政策等。但发展3D打印这项新兴技术，凭借热情是不够的，3D打印产业链中，关键技术、材料、软件的制约有待突破，3D打印技术现在还处在初级阶段，如果企业和资本大量涌入，短时间内不能产生效益。

  1. 这段文字意在说明：
  2. A.3D打印的应用范围有待充分挖掘
  3. B.3D打印技术的发展遭遇瓶颈
  4. C.应出台政策鼓励3D打印技术的发展
  5. D.资金不宜盲目涌入3D打印领域
  1. 文段首先描述3D打印的火热现状，接着通过转折词“但”指出3D打印目前存在的问题，最后通过反面论证提出对策，指出“如果企业和资本大量涌入，短时间内不能产生效益”，即强调现在资金不宜大量涌入3D打印领域，对应D项。
  2. A项：“应用范围”在文段中没有被提到，属于无中生有的表述，排除；
  3. B项：该项为现状和问题的表述，非重点，排除；
  4. C项：“应出台”表述错误，文段中各地已出台各项优惠政策，且“政策”为转折前的表述，非重点，排除。
  5. 故正确答案为D。

例10：（2020浙江）数字经济的发展与个人信息保护本不应是非此即彼的选择。然而， 无论是传统产业的兴衰，还是近年来经济新业态的更迭，许多教训在提醒我们，发展与规范极易顾此失彼。大数据是数字经济的“粮食”，而数字经济则被看作是中国经济实现弯道超车的良机。然而，数字经济与信息数据保护之间的冲突已日益突显。个人信息保护基本规范缺位，监管执行无力且迟缓，平台侵权频发，诸多难题倘若不能尽快破解，最终必然拖累数字经济本身。

  1. 这段文字意在强调：
  2. A.中国发展数字经济须先建立规范有序的市场
  3. B.数字经济时代依法依规保护个人信息迫在眉睫
  4. C.数据收集与个人权益保护之间的矛盾难以调和
  5. D.迅猛发展的数字经济不可避免地带来数据安全问题
  1. 首先提出数字经济发展与个人信息保护应并行的观点，接着通过“然而”转折指出发展与规范易顾此失彼，再次转折强调数字经济与信息数据保护的冲突日益凸显，最后通过反面论证强调解决相关问题的重要性。
  2. B项：是针对文段中数字经济与信息数据保护冲突问题提出的解决问题的合理对策。
  3. A项：对策无中生有，文段重点强调的是个人信息保护的重要性，并非建立规范有序的市场。
  4. C项：只是对问题的表述，并非文段重点，文段重点在于强调解决问题。
  5. D项：同样是对问题的表述，不是文段重点，文段重点是要解决数字经济与个人信息保护的冲突。
  6. 因此，选择B选项。

例11：（2018民生银行）作为整体，中国在世界上举足轻重；但作为个人，不少中国人还觉得自己一无所有。国家之强和个人之弱使一些人心理失衡，觉得自己活得还是像在半殖民地时代受人家欺负的受害者。正因如此，我们更需要对自己生存的状态有理性的认识，克服狭隘的“受害者情绪”。否则，崛起的中国将难以担当与自己的国际地位相称的责任。

  1. 这段话谈论的核心意思是（ ）。
  2. A.中国急需提高国民的个人地位
  3. B.中国人需要调整自己的心理状态
  4. C.中国人为什么有“受害者情结”
  5. D.崛起的中国要承担相应的国际地位
  1. 文段首句讲了中国目前国强民弱的现状，在这样的情况下中国人自己觉着自己是受害者。接着，用“正因如此”引导了作者的观点是应更需要对自己生存的状态有理性的认识。
  2. 最后一句话由“否则”引导的反面论证。依据我们学习的反面论证法的技巧，有观点直接看前面的观点。
  3. 所以本文段的主旨句应为“我们更需要对自己生存的状态有理性的认识，克服狭隘的‘受害者情结’。”对这句话同义替换之后，得出答案B。
  4. A项属于无中生有，故A项错误。C项出现在文段结果部分，这个部分不是旨在讨论中国人有这种情节的原因，而是要克服我们就要有理性的认识，故C项错误。D选项也是出现在因果关联词果的部分，只是文章中崛起的中国只有在前面的前提成立下，才可以实现，故D项错误。所以答案选B。

并列关系 ​

字数: 5242 字 时长: 20 分钟 VIP文档

  1. 两个或两个以上的词、词组、句子，具有同样的语法，同时出现在一句中。它们之间相互平行，地位平等，没有主次之分。但在排列时，需要考虑一定的逻辑顺序，如从重要到次要、从复杂到简单、从社会到个人等。

一、并列 ​

  1. 1、表示并列关联词：一边……一边……、既……又……、一方面……另一方面……、那么……那么……

  2. 2、表示并列的词语：而、又、也、还、同时、另外、此外、与此同时。“而”可以表示并列，也可以表示转折

  3. 3、隐性的并列关系：

    1. （1）材料当中的结构一致。例如第一句话是“有一种情况是……”，第二句话是“还有一种情况是……”。
    2. （2）并列主体。例如第一句话是“国家………”第二句话是“社会………”第三句话是“个人…”。
    3. （3）按照时间顺序展开。例如第一句话是“古代...”，第二句话是“近代..."。

  4. 4、表示并列的符号：

    1. （1）顿号：顿号用于分隔句子中同类的词语或短语，表示较短的停顿。一般表示平行并列。比如他买了苹果、香蕉、橙子。
    2. （2）分号：分号用于分隔较长的并列成分，尤其是当这些成分内部已包含逗号时。比如他去了北京，参观了故宫；去了上海，游览了外滩；去了广州，品尝了美食。

  5. 5、按照一定顺序展开：

    1. （1）空间顺序。如介绍建筑时先写外部布局再写内部结构。
    2. （2）时间顺序。如按年代或朝代排列，"唐朝、宋朝、元朝"。

二、解题技巧 ​

  1. 1、并列关系中，重要性前后一致，例如“既美丽又善良”，不是单独强调美丽，也不是单独强调善良，这是表述片面，即只提及其中一个方面，这种选项是以偏概全的，排除。而是美丽和善良同样重要。
  2. 2、若文段涉及的并列论述的是不同话题，此时需要全面概括；
  3. 3、要想概括多个方面，需要用一些表述宽泛、笼统的词语，如：许多、一些、不同、各种等，才可以概括全面，也是正确选项的特征。

三、随笔练习 ​

例1：（2019国考）基础数学是一门对天赋要求极高的学科，它的高度抽象性让不具备这种天赋的人望而生畏。在某种意义上可以说，是数学选择了它的追随者，而非相反。加之数学是一门完全依赖人自身最纯粹的大脑机能进行探索的学科，这使得一流的数学研究介乎学问和艺术创造之间，总是在“灵感乍现”的时刻产生突破。因此，数学家实际上是一个极其冒险的职业，其成就几乎完全仰仗天赋和灵感的偶然眷顾。另一方面，对具有数学才能的人来说，现代社会充满了机会的诱惑，金融、计算机、互联网，都是比数学研究更赚钱的行业。

  1. 这段文字意在：
  2. A．解释数学家可遇不可求的现象
  3. B．说明天赋对于数学研究的意义
  4. C．探讨基础数学研究的本质规律
  5. D．强调基础数学发展面临的困境
  1. 文段开篇指出“基础数学”对天赋有较高的要求，并指出是数学选择了它的追随者，随后通过并列关联词“加之”指出“数学”研究依赖大脑机能进行探索，往往在“灵光乍现”的时刻产生突破。
  2. 后文通过结论词“”进行总结，并通过并列关联词，故文段重点围绕“数学家”这一话题展开，强调能成为“数学家”的人较少，可以遇到但不可强求，对应A项。
  3. B项，“天赋对于数学研究的意义”对应“因此”后的并列的部分内容，片面，且未提及核心话题“数学家”，排除；
  4. C项，“本质规律” 无中生有，文段并未提及，排除；
  5. D项， “基础数学”偏离文段核心话题，文段重点强调“数学家”这一核心话题，排除。
  6. 故答案为A。

例2：（2021广东选调）今后一个时期是中国实现绿色转型的攻坚期。加快推进绿色转型，才能实现生态环境根本好转，为美丽中国建设目标的实现创造条件。与此同时，中国绿色转型的成功探索，也将为广大发展中国家提供绿色发展模式的新选择，为构建可持续发展的人类命运共同体贡献中国智慧和中国方案。

  1. 上述文段主要说明的是：
  2. A.我国实现绿色转型的主要目标
  3. B.我国实现绿色转型的重要意义
  4. C.我国实现绿色转型的主要内容
  5. D.我国实现绿色转型的基本途径
  1. 文段开篇引出“”的话题，接下来通过“加快推进绿色转型，才能……”实现绿色转型对生态环境好转、实现美丽中国的促进作用。
  2. ，后文“也将为……贡献中国智慧和中国方案”体现我国绿色转型对广大发展中国家、对世界的重要贡献，故文段，对应B 项。
  3. A 项“主要目标”、C 项“主要内容”、D 项“基本途径”，均无中生有，排除。
  4. 故正确答案为 B。

例3：（2010国考）在安科莱，以畜牧为生的希马人和以农业为生的伊鲁人共同居住；在亚利桑那，纳瓦霍人以前靠狩猎和采集为生，现在主要以畜牧为主，他们与经营农业的霍皮人为邻；澳洲东南沿海地带以前住着以狩猎和劫掠粮食为生的土著居民，现在却住着从事农业、畜牧业及工业的欧洲人。

  1. 作者列举这些事实意在说明：
  2. A.环境迫使人们接受某种生活方式
  3. B.人们对自然环境有很强的适应能力
  4. C.不同文化的族群完全有可能和谐相处
  5. D.地理环境并非人类生产方式的决定因素
  1. 根据提问“意在”可知本题为中心理解题。文段中的三个分号标志的典型并列文段，列举了几种在同一地理位置生活着不同生产方式的人群的情况。需要从每个分句中提取共性。
  2. 通过阅读可知，每个分句都在强调同样的地点可以有不同的生产方式，结合选项，D项为此共性的同义表述。
  3. C项为迷惑项，文段最后一个分句“以前住着……现在却住着……”不能说明“不同族群和谐相处”，故而排除。A、B两项内容，文段中没有提及，排除。
  4. 故正确答案为 D。

例4：（2018事业单位联考）《史记》的生命力源于历代读者对它的接受。它作为一条文化链，有传播者、接受者、研究者，历代不绝，乃至于形成一门学问——“史记学”。这门学问的形成过程，就是《史记》生命力延伸的过程。汉魏六朝是传播和初步研究时期，此后不断发展，20世纪以来进入高潮时期。《史记》的生命力还来源于其世界性的传播与研究：在日本，各种形式的《史记》抄本、刻本，或选本，或全本，数量在百种以上，普及程度非常广泛；在欧美国家，《史记》也愈来愈受到重视，全本的俄语、法语版已经问世，全本的英语版也即将完成。世界范围的《史记》研究愈来愈深入，显示出《史记》永久的魅力和生命力。

  1. 这段文字主要介绍了：
  2. A.《史记》的传播过程
  3. B.“史记学”的独特魅力
  4. C.《史记》受重视的原因
  5. D.《史记》生命力的来源
  1. 由“《史记》的生命力源于······《史记》的生命力还来源于······”可知文段为并列结构，需要将并列前后分句的内容进行全面概括。
  2. 文段前半部分介绍《史记》的生命力源于历代读者对它的接受，接下来具体说明历代读者是如何使其形成一门学问并进行传播的；
  3. 文段后半部分介绍《史记》的生命力还来源于其世界性的传播与研究，接下来举例说明《史记》在全世界不同国家的传播，体现出其生命力。
  4. 故整个文段重在介绍《史记》生命力来源的两个方面，对应D项。
  5. A项“《史记》的传播过程”、C项“《史记》受重视的原因”均对应并列后半部分的内容，表述片面，排除。
  6. B项：“‘史记学’的独特魅力”对应并列前半部分的内容，表述片面，且对应解释说明的部分，非文段重点，排除。
  7. 故正确答案为D。

例5：（2019江苏）从顶层设计上看，我国工匠精神培育建设需立足于搭建协同保障平台、开展不同主体间的多元对话、建构产教研协同发展机制以及强化职业院校精神培育功能。从协同发展的视角看，我国工匠精神的培养主体亟须根据自身的角色定位与职能属性，强化市场管理制度，培育高品质消费市场，不断完善技术技能人员薪酬体系，构建职业院校、实训中心、企业三站式互动机制。

  1. 这段文字意在说明
  2. A.我国工匠精神培育的主要策略
  3. B.我国工匠精神培育的顶层设计
  4. C.必须强化工匠精神培育工作的管理
  5. D.搭建协同平台是工匠精神培育重心
  1. 文段开篇“从顶层设计上看”论述了我国工匠精神培育建设的做法，之后“从协同发展的视角看”论述了我国工匠精神培育主体亟需做什么，故文段为并列结构，从两个角度论述了我国工匠精神培育当前应该如何做，对应A项，当选。
  2. B项“顶层设计”和D项“搭建协同平台”，均仅为文段“从顶层设计上看”的内容，表述片面，排除；
  3. C项，“强化管理”为“从协同发展的视角看”的做法，仅为对策的一方面，表述片面，排除。
  4. 故正确答案为A。

例6：（2018事业单位联考）青霉素被发现以前，细菌是人类历史上制造肺炎、伤寒、痢疾、肺结核等疾病的最强大杀手。50多年前，人类身体内部拥有包括细菌在内的巨大的微生物群落的概念才开始流传。人们逐渐尝试认识，我们体内的这些定居者们究竟如何与我们互动？在敌对之外，我们与这些定居者们是否还存在另一种亲密关系？而现在，学术界流行的是“我们只有是人类”的说法，指每10个构成人体的细胞中，就有9个是搭便车的细菌。人体微生物已经成为热门话题，攻占了顶尖学术期刊的无数版面。

  1. 对这段文字概括最恰当的一项是：
  2. A.学界对细菌的研究已达到了高峰
  3. B.细菌对人体的作用不能一概而论
  4. C.多年来人类对细菌的认识一直存在误区
  5. D.人们对人类与细菌关系的认识不断更新
  1. 文段开篇指出在青霉素被发现以前，细菌是制造很多疾病的杀手；
  2. 接着论述在 50 多年前，人们开始认识到人体内部也存在很多细菌，并提出我们如何与细菌进行互动等问题；最后通过“而现在”引出如今人们对于细菌和人类关系的认识又有了新的变化，即人体中有90%的细胞中都由细菌构成。
  3. 故文段通过“青霉素发现前”“50 多年前”“现在”三个时间顺序标志词引导的并列结构，分别阐述了人类在不同的时期对于人类与细菌关系的认识。对文段进行归纳概括，对应 D 项。
  4. A 项：“已达到了高峰”文段未提及，文段只是论述当前对微生物的讨论是个热门话题，“高峰”为无中生有，排除。
  5. B 项：“细菌对人体的作用”不是文段强调的重点，文段谈论的是人们对于人类和细菌关系的认识，排除。
  6. C 项：“一直存在误区”表述过于绝对，文段只是强调人类对于细菌的认识不断更新，而非一直以来对细菌的认识都是错的，排除。
  7. 故正确答案为 D。

例7：（2022山东）受全球气候变化影响，北极航道夏季通航将成为现实，而跨越北极上空是连接世界很多国家的最短距离，俄、加、美、挪等国相继成立北极安全部队，俄罗斯宣布将全面恢复苏联在北极地区遗留的13个航空基地，周边国家对北极的争夺加剧。随着空间技术发展，太空领域也正在成为军事大国博弈的制高点，高超声速武器、天基武器系统的研发与装备，今年可能进一步加速。此外，网络虚拟空间成为军事博弈的前沿，一些国家已经或计划将网络作战力量提升到与陆海空常规作战力量、特种作战力量同等重要的地位。

  1. 这段文字意在说明：
  2. A.许多大国在新战略空间展开博弈
  3. B.大国抢占地球空间领域发展先机
  4. C.网络虚拟空间作战力量正逐步提升
  5. D.全球气候变化对大国战略产生影响
  1. 文段开篇指出受气候变化影响，周边国家对北极空间的争夺加剧，之后通过并列引导词“也”指出太空领域正在成为军事大国博弈的制高点。
  2. 最后通过并列引导词“此外”指出网络虚拟空间成为了军事博弈的前沿，网络作战力量的地位进一步提升。故文段为“也”“此外”引导的三方面并列，全面概括，对应A项。
  3. B项，“地球空间领域”仅对应“此外”前的内容，表述片面，排除；
  4. C项，“网络虚拟空间”仅对应“此外”后的内容，表述片面，排除；
  5. D项，“气候变化”对应“也”前的背景，非重点且表述片面，排除。
  6. 故正确答案为A。

例8：（2021上海）淮河上游，在距今8000多年前的贾湖遗址的一些红烧土块内发现了保存很好的稻壳印痕，经过鉴定，这些水稻属于人工栽培而非野生；淮河中下游地区，距今8000余年前的顺山集遗址、距今7000多年前的双墩遗址、距今6000多年前的龙虬庄遗址和距今5000年前的尉迟寺遗址等，都出土了人工培育的水稻。

  1. 这一语段说明了：
  2. A.淮河流域最晚在新石器中期便已开始种植人工培育的水稻
  3. B.考古发现淮河上游的历史早于淮河中下游
  4. C.淮河流域比长江流域更适合种植水稻
  5. D.考古证明，野生水稻并不存在
  1. 文段首先介绍了淮河上游发现了距今8000多年前人工栽培的稻壳印痕，接着指出淮河中下游地区多个遗址也出土了人工培育的水稻，故文段为并列结构，均在介绍淮河流域历史上存在人工培育的水稻的事实，对应A项。
  2. B项，缺少主题词“人工培育的水稻”，排除；
  3. C项，文段未把“淮河流域”与“长江流域”进行对比，属于无关对比，排除；
  4. D项，文段主要论述的是“人工培育的水稻”，而非“野生水稻”，主题词错误，排除。
  5. 故正确答案为A。

例9：（2020新疆35%）拿破仑去世后，其尸检报告给出的拿破仑官方身高是5尺2寸（法寸），即1.68米左右。按照现在的标准来看并不算高，但有历史学家估计，十八世纪末、十九世纪初，法国人的平均身高大致为1.66米至1.67米。拿破仑是矮个子的说法其实与英法两国间的国际政治相关。1803年《亚眠和约》破裂之后，英国媒体把拿破仑5尺2寸的身高写在了报纸上，画在了漫画中，却没有标明其“法寸”的长度单位。英国读者便用英寸来理解拿破仑的身高，换算为1.57米。

  1. 这段文字意在说明：
  2. A.说拿破仑是个矮个子纯属误解
  3. B.其实拿破仑比一般法国人要高
  4. C.英国人污蔑拿破仑是矮个子
  5. D.要用历史眼光看拿破仑的身高
  1. 整个文段为并列结构，前半部分通过介绍法国人的平均身高为1.66米至1.67米，而拿破仑官方身高是5尺2寸（法寸），即1.68米左右，说明在当时拿破仑的身高并不矮。后半部分阐述拿破仑矮个子的说法是因为英国媒体把拿破仑5尺2寸的身高写在了报纸上，却由于政治因素没有标明其“法寸”的长度单位，被误以为身高矮，故文段重在强调讨论拿破仑的身高应该考虑其所处的时代背景。对应D项。
  2. A项，拿破仑个子矮被误解仅对应文段前半部分，表述片面，排除；
  3. B项，拿破仑比一般法国人高仅对应文段前半部分，表述片面，排除；
  4. C项，英国人污蔑拿破仑是矮个子仅对应文段后半部分，表述片面，排除。
  5. 故正确答案为D。

例10：（2015江苏）由于现今大学对自身职能以及职能履行的认识存在一定的偏差，大学内部制度中缺乏教学的价值体现，教师在教学方面的努力、贡献难以得到应有的承认，这样就桎梏了教师的教育热情，直接影响教学质量与大学的社会公信力。另外，大学及教师群体对“学术”内涵的窄化理解使得教学与科研相对立，未将教学上升到学术的层面加以理解。在大学中广泛存在着对专业科研的学术认同，而作为知识传播的教学则难以进入学术之列。

  1. 这段文字主要强调的是：
  2. A.大学应提高对自身职能的认识
  3. B.大学教学质量有待进一步提高
  4. C.大学必须提高对教学价值的认识
  5. D.大学内部应对教学与科研一视同仁
  1. 文段为以“另外”衔接的并列结构，首先论述了大学内部制度缺乏教学的价值体现，即强调应对教学予以认可和重视；“另外”后论述了轻视教学导致教学与科研相对立的尴尬局面，故总结两个并列句子全面概括可知，文段主旨应为呼吁大学提高对于教学的重视，即C项。
  2. A项“自身职能”为第一个并列分句中的原因部分，表述片面；B项“教学质量”是轻视教学后产生的影响，为次要信息，故排除A、B两项；D项，讨论的是科研与教学之间的关系，为第二个并列分句内容，表述片面，排除。
  3. 故正确答案为C。

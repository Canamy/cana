推断下文 ​

字数: 5057 字 时长: 19 分钟 VIP文档

  1. 1、考查频次：一般，考试中考察题目数量在1-2道题左右。
  2. 2、题型特征：根据已知文段推断下文内容。
  3. 3、提问方式：
    1. （1）接下来最可能讲的是？
    2. （2）这段文字之后作者最可能阐述/说明的是？
    3. （3）接下来最可能讲的/讲述的是？
    4. （4）根据这段文字，接下来会说明？

解题思路 ​

  1. 1、 原文中论述过的内容，下文不会再出现。比如原文出现XXX的意义，则下文（答案）不会再谈论XXX的意义

  2. 2、 下文内容应与原文话题一致，不会突然跳跃到无关信息。

  3. 3、下文推断常见结构:

    1. （1）上文文末给出一个新内容，下文很可能会解释这个新内容。
    2. （2）上文文末给出一个现象，下文很可能围绕这个现象展开，或者解释“为什么”会产生这个现象。
    3. （3）上文文末提出问题，下文很可能分析问题/解决问题。
    4. （4）上文文末给出宏观对策，下文给出具体对策。

  4. 4、下文推断不易识别结构:

    1. （1）时间对比：过去不行 → 现在有了新突破。
    2. （2）总分：上文是总+分 → 下文补充另“分”。
    3. （3）转折关系：然而、但是、尽管 → 下文可能提出相反观点。
    4. （4）举例说明：例如、比如、以…为例 → 顺着观点往下说。
    5. （5）总结结论 → 通常无需展开，下文可能结束(需结合选项判断)。

  5. 随着近年来题目的难度升级，也有少部分题目的陷阱设置较多一些，当文段尾句谈论的话题比较多难以判断落脚点时，就需要行文脉络找重点的方法，判断出文段尾句重点想要强调的话题。

随笔练习 ​

例1：（2019国考）潜水员在执行水下任务的过程中，普遍采用信号绳作为主要通信工具，即通过对信号绳的拉、抖组成系列信号来实现对陆上的简易通信。这种通信方式便捷、直接，但是其弊端也是显而易见的：信号绳仅能实现有限信息量的表达，且信号传输过程极易受复杂海水环境影响而中断或失效，带来安全隐患。2015年，就曾有潜水员的信号绳被缠住而险些发生事故。可以说，潜水员在执行水下任务时，是真正的命悬一 “线” 。针对信号绳的诸多弊病，结合智能穿戴设备在民用领域的快速发展，面向军事潜水领域的智能穿戴产品逐渐成为科技工作者的研发热点之一。

  1. 这段文字接下来最可能讲的是：
  2. A. 军事潜水领域智能穿戴设备的关键技术
  3. B. 信号绳在军事领域传递信息中的缺陷
  4. C. 日常生活中智能穿戴设备的发展现状
  5. D. 人工智能技术引入穿戴设备的前景预期
  1. 重点分析尾句。尾句针对上文潜水的弊端以及安全隐患等问题提出了对策，指出 “针对信号绳的诸多弊病，结合......逐渐成为科技工作者的研发热点之一” 可知，尾句重点强调的是智能穿戴设备对军事潜水领域的重要性。
  2. 分析选项。A项提到了军事潜水以及智能穿戴，和尾句话题一致。B项信号绳的缺陷在上文已经论述，排除。C项是智能穿戴设备在日常生活中的发展现状，而尾句强调的是对军事潜水领域的作用，偏离尾句话题，排除。D项“前景预期”过于笼统，且没有提到军事潜水领域，排除。
  3. 因此，选A。

例2：（2014深圳）传统的直接连接存储方式完全以服务器为中心，通过线缆直接连接到计算机处理器，并寄生在服务器或客户端上。随着应用系统的扩大和数据量的增长，随着用户对数据服务的要求不断提高，存储方式由传统直连式存储走向分离式存储。

  1. 由此可推知下文将要论述的是（ ）。
  2. A.“分离式存储”的弊端
  3. B.“直连式存储” 的弊端
  4. C.“分离式存储”将满足用户的需求
  5. D.“分离式存储”的含义和形式
  1. 文段尾句首次提到“分离式存储“这一概念，前文没有提到分离式存储，根据话题一致的原则，下文内容应与此相关，首先排除B项。
  2. 依据一般逻辑，提出概念后，应先就说其“是什么”展开论述，故选D项。

例3：（2014河北）临近年底，部分二三线城市楼市调控也走到了一个微妙时点。业内人士认为，目前来看，一些二三线城市在限购政策落实方面“疲态”渐显，或试图放松限购令，或期待限购令自动到期作废。截至2013年底，多个限购令将到期的城市未就限购令走向表态。

  1. 根据上述文字推断，作者接下来最有可能讲述的内容是（ ）。
  2. A.限购令对楼市的影响
  3. B.对中国楼市现状的分析
  4. C.二三线城市对限购令的态度
  5. D.限购令到期后城市楼市走向
  1. 文段尾句提出一个现象：截至2013年底，多个限购令将到期的城市未就限购令走向表态。那么接下来作者最有可能围绕这个现象展开说明，C项与此相符。
  2. A项为限购令对楼市的影响，而文段侧重于城市的态度，排除A项；
  3. B项没有讲到主题词 “限购令” ，排除B项；
  4. D项为强干扰项，文段最后一句的主体是“城市”，而D项的主体是“楼市”，可以排除。
  5. 故正确答案为C。

例4：（2022重庆选调）冬季是心血管疾病的好发季节。相关研究数据表明，每年的12月至次年3月是心血管病发病高峰期，发病数明显高于其他月份，这与血脂季节性波动的结论也相一致。据统计数据显示，因心血管疾病死亡的事件中约15.8%与天气寒冷相关，而仅有1.3%与天气炎热相关。心力衰竭、心肌梗死和脑卒中在寒冷季节的发病率和死亡率均高于温暖季节。

  1. 这段文字接下来最有可能讲的是：
  2. A.心血管疾病为何冬季高发
  3. B.心血管疾病的症状有哪些
  4. C.如何有效治疗心血管疾病
  5. D.如何预防冬季心血管疾病
  1. 文段开篇引出话题，冬季是心血管疾病的好发季节的问题，随后通过数据资料具体论述冬季心血管病的高发情况。
  2. 根据行文脉络提出问题——分析问题，故接下来应论述冬季心血管疾病为何高发，对应A项。
  3. B项“心血管疾病的症状”、C项“治疗心血管疾病”均与文段核心话题不一致，文段核心话题为“冬季”“心血管疾病”，不可能为接下来要论述的内容，排除；
  4. D项“如何预防”为针对心血管疾病冬季好发的对策，按照逻辑顺序，应先分析原因，再提出对策，故D项应在A项之后论述，排除。
  5. 故正确答案为A。

例5：（2017国考）在文字还不普及的时代，民间故事承担了培养人生观、道德观、伦理观的职能，听故事是人们学习传统文化、自然知识、人生哲学等的重要渠道。然而，飞速发展的现代科技正在深刻改变着人们的生活方式，也极大地影响着人们的接受习惯和审美趣味。当下的年轻人对传统文化的感知和了解，更多的是通过影视作品、网络小说、电子游戏等途径，年轻人对传统民间故事中的一些经典形象越来越陌生，不少专家学者表示，打捞 “失落” 的民间故事刻不容缓。

  1. 接下来最可能讲的是（ ）
  2. A.让民间故事为现代人接受的途径
  3. B.民间故事与传统文化之间的关系
  4. C.年轻一代对民间故事的了解状况
  5. D.现代科技对民间故事传播的冲击
  1. 文段尾句提出一个宏观对策，指出对民间故事进行打捞刻不容缓。
  2. 按照正常逻辑可知，，对应A项。
  3. B项 “民间故事与传统文化之间的关系” 并非尾句谈论的重点，排除。
  4. C项 “年轻一代对民间故事的了解状况” 和D项 “现代科技对民间故事传播的冲击” 为前文已经提过的内容，均排除。
  5. 因此选A。

例6：（2019辽宁30%）在人人皆可言教育的年代，猜疑与隔阂正在消解传统意义上的师道尊严。教师，也算是一个清苦的职业。看似有寒暑假，实际上多数时间都用来备课、评课；双休日，学校要承担各类社会考试；工作日，课多人少，一人请假就可能影响教学进度。在这样的工作压力下，倘若再得不到应有的尊重，轻则让教师心灰意冷，重则令名师远走高飞。

  1. 接下来最有可能讲述的是：
  2. A.如何消除社会对教师的猜疑与隔阂
  3. B.涵养师道尊严的途径
  4. C.造成社会对教师误解的原因
  5. D.呼吁全社会尊重教师
  1. 尾句通过反面论证提出对策，即教师应该得到充分的尊重。
  2. 下文谈论的内容应与文段尾句话题保持一致，且逻辑一致。故下文应具体说 "教师应该得到充分的尊重" 具体途径和措施，对应B项，当选。
  3. A项“猜疑与隔阂”为首句谈论的话题，下文不会继续论述，排除；
  4. C项“对教师误解”话题错误，文段最后谈论的是“对教师的尊重”，排除；
  5. D项“呼吁全社会尊重教师”为尾句反面论证的内容，即为尾句的同义替换，而下文应谈论让教师得到尊重的具体途径和做法，排除。
  6. 故正确答案为B。

例7：（2019广西桂林事业单位）河的两岸风光旖旎。西岸，草原一望无际，绿色的波涛逶迤而去，在天际同蓝天连成一片。三四千头成群的野牛在广阔无垠的草原上漫游。这时，一头年迈的野牛劈开波涛，游到河心小岛上，卧在高深的草丛里。看它头上有两弯新月，看它沾满淤泥的漂浮的长髯，你可能把它当成河神。它踌躇满志，望着那壮阔的河流和繁茂而荒野的两岸。

  1. 作者接下来最有可能着重介绍的是（ ）。
  2. A. 叙述“河神”野牛的传说
  3. B. 描写草原上的其他野生动物
  4. C. 描写位于河岸上自己的家乡
  5. D. 描写河另一岸的美丽风光
  1. 本题为接语选择题，重点看尾句。文段开篇提出河的两岸风光旖旎，随后论述西岸风光，具体描绘了西岸的草原、草原上的野牛、野牛的神态，文段尾句在描绘“野牛”的神态，本质上属于“西岸风光”，结合行文脉络，文段为总分结构，总句提及河的两岸，分述句应先论述河的西岸风光，，故下文应论述河另一岸风光，对应D项。
  2. A项，野牛并不是“河神”，文段论述“你可能把它当成河神”，为作者的写作修辞手法，故选项表述有误，排除；
  3. B项，“其他野生动物”文段并未提及，为无中生有，排除；
  4. C项，“自己的家乡”文段并未提及，为无中生有，排除。

例8：（2018山东）规模化是工业现代化的重要方式之一，中国农业发展是否适应大规模农业生产模式还值得商榷。美国多年的大农场生产方式已经证明，集约化的农业生产会造成农业生态环境退化、生物多样性被破坏，进而影响到粮食安全。当前，我国人均耕地面积仅1.52亩，只有世界平均水平的，更远低于美国的人均水平，所以美国的大农场模式并不一定适合中国的发展。同时，我国农业现代化起步较晚，相较于美国等发达国家，无论是在农业科技还是经营管理方式上还存在一定差距，现代化的“步子”骤然迈得太大，很有可能造成上下层“脱节”。

  1. 这段文字接下来最可能说的是：
  2. A.农业改革不应操之过急，而应稳扎稳打
  3. B.发展大规模农业生产模式应吸取国外教训
  4. C.农业生产的规模化应量体裁衣、因地制宜
  5. D.生产环节的脱节是农业规模化发展的瓶颈
  1. 文段开篇指出中国农业是否适应“规模化”生产有待商榷，即提出问题，接着指出美国进行大规模生产带来的消极影响，并由此强调这种大农场模式不适合中国的发展，并通过“同时”引导的并列关系，对这一问题继续进行分析，故整个文段为提出问题—分析问题的结构。
  2. 按照行文脉络下文应着重解决问题，即提出对策，即农业生产的规模化应结合具体情况，因地制宜，对应C项。
  3. A项“农业改革”、D项“生产环节的脱节”均与文段的核心话题“大规模农业生产”不一致，排除；
  4. B项“吸取国外教训”对应“美国的大农场模式并不一定适合中国的发展”，前文已详细阐述过，排除。
  5. 故正确答案为C。

例9：（2017河南32%）大家都知道中国文化需要“走出去”，但是很少有人清楚地知道文化方面什么东西要走出去。正如商家做营销，你要推销产品，首先必须有产品。营销只是包装和策略问题，营销得当就能够逐步确立人们对产品的认同感。但无论如何，首先必须有高质量的产品。如果产品质量低下，营销做得再好也无济于事。

  1. 这段文字接下来最可能讲的是（ ）
  2. A．营销与产品的相互关系
  3. B．如何建立文化产品的质量标准
  4. C．目前中国文化“走出去”所面临的困境
  5. D．中国文化“走出去”应确立的主要产品
  1. 文段首先指出，很少有人知道中国文化什么东西应该“走出去”，接着以“营销”为例进行详细说明，指出想营销必须先有产品，且需要高质量的产品，故文段强调的是中国文化想要“走出去”应明确“走出去的具体内容”，下文最可能围绕“走出去”的“产品”展开论述，对应D项。
  2. A项，“营销与产品”仅为文段中的例子，其目的是为了强调“走出去”的中国文化需明确“产品”，故与文段核心话题衔接不当，排除；
  3. B项，未提及“中国”这一话题，且“质量”非重点，文段重在强调应确定“中国文化走出去的产品”，排除；
  4. C项，“困境”无中生有，与文段衔接不当，排除。

例10：（2017山东）中国的古代城市都有城墙吗？在人们以往的印象中，古代的城市似乎必然有城墙，尤其是都城，高耸的城墙彰示着皇权的至高无上，城墙的失守往往意味着帝国的末日。明清北京城、元大都、北宋汴梁城、隋唐长安城与东都洛阳城······这些城内的里坊格局，外围的高大城郭，构成了帝国都城最鲜明的物化表征。

  1. 这段文字接下来最有可能讲的是：
  2. A.帝国都城的发展演变
  3. B.古代都城城墙的作用
  4. C.皇权与城市格局的关系
  5. D.没有城墙的中国古代城市
  1. 文段开篇提出疑问“中国的古代城市都有城墙吗？”显示出作者对此的怀疑态度，接着提到“似乎必然有城墙”亦反映了作者对此的不确定，接下来介绍了都城都有城墙并进行了详细解释说明。
  2. 后文应顺着观点“中国的古代城市都有城墙吗？”，先解释了有城墙的城市，后文应具体描述不存在城墙的城市，与作者态度保持一致，故D项衔接得当，当选。
  3. 文段都在围绕“城墙”论述，故排除未提及“城墙”的A、C两项。
  4. B项“古代都城城墙的作用”在文段中已有体现，即“彰示着皇权的至高无上”，故后文不会再次论述，排除。
  5. 故正确答案为D。

例11：（2021黑龙江）目前，由于支持行业企业办学的财税优惠政策缺乏实施细则，激励不够，行业企业举办职业教育面临诸多实际困难，举办职业教育和培训的功能正在被弱化。国有企业曾经是举办职业教育的重要力量，在国企改革中，一些职业院校被视为不良资产列为剥离对象，经费支出没有对应的支出科目，公共财政拨款和相应的政策支持没有渠道。同时，不少社会力量办职业教育发展也面临民办教育和职业教育的双重困境与双重压力，毕业生普遍存在就业、用人等歧视，社会认同度低、吸引力较差。

  1. 这个文段接下来最有可能讲述的是：
  2. A.如何办好高质量的职业院校
  3. B.国家鼓励行业企业的职业教育
  4. C.国有企业应依法履行实施职业教育的义务
  5. D.校企合作是现代职业教育的发展方向
  1. 文段开篇介绍了目前行业企业举办职业教育面临着诸多问题，接着通过并列展开说明问题，即一方面，国有企业经历了国企改革，一些职业院校失去了公共财政拨款和相应的政策支持；另一方面，社会力量办的职业教育也面临着双重困境和压力，其毕业生面临着就业歧视、社会认同度低等问题。
  2. 故前文主要介绍举办职业教育面临的具体问题，接下来应围绕解决问题展开，即接下来应介绍办好职业教育的具体对策，对应A项。
  3. B项“国家鼓励”、C项“依法履行实施职业教育的义务”、D项“校企合作”文段均未提及，衔接不当，排除。
  4. 故正确答案为A。

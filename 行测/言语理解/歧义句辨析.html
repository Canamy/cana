歧义句辨析 ​

字数: 2272 字 时长: 9 分钟 VIP文档

  1. 深圳、上海、吉林、山东等省市曾考查歧义句，难度一般，需逐条分析。

一、歧义句的类型 ​

歧义句是指一个句子在特定语境下可能被理解为两种或多种不同的含义，导致语义不明确的现象。歧义的产生通常源于词汇、语法结构、语境或语音（如多音字）的模糊性。

  1. 1、语汇歧义

    ：词有不同的词性，可做名词或者动词；词语本身有多个含义；最后导致句子意义不确定。

    1. 例1：我们家门没锁。可理解为“门没有上锁”；也可理解为“门没有锁头”
    2. 例2：我要炒肉丝。可理解为“炒”这个动作，也可理解为“炒肉丝”这个菜
    3. 例3：他原来住这里。“原来”可理解为“以前”，也可理解为“竟然”

  2. 2、语法歧义：

    1. 例1：放弃美丽的女人让人心碎。让人心碎的是“女人”还是“放弃美丽的女人”这件事。对象不明
    2. 例2：某高层嘱咐几个法院的领导，一定要让这起案件销声匿迹。可理解为“几个领导”，也可理解为“几个法院”。修饰不清
    3. 例3：老王的笑话讲不完。可理解为“他能讲的笑话”，也可理解为“关于他的笑话”。关系不清
    4. 例4：老李看到老王带着他的老婆出去了。不知道是老李的老婆还是老王的老婆。指代不明
    5. 例5：上岸村通知说，让村民本月15日前去汇报。“前去”可理解为“15日之前去”，也可理解为“15日当天前往”。语义关系不明
    6. 例6：鸡不吃了。到底是鸡不吃（东西）了还是（人）不吃鸡了。施受关系不明：动作的发出者（施事）和承受者（受事）关系不清。

  3. 3、停顿歧义：停顿在不同的位置具有不一样的意思。

    1. 例1：山上的水，我们留给晚上来的人喝。晚/上来的/人，指“山上晚了的人”；晚上/来的/人，指“夜晚上山的人”
    2. 例2：发展经济学的理论渊源是人类关于推进社会经济快速发展的思想。发展/经济学的理论渊源，指发展“经济学”学科的理论；发展经济学/的理论渊源，指“发展经济学”这门经济学的理论
    3. 例3：不适当地浇水施肥。是“不/适当地浇水施肥 （完全不做）”还是“不适当地/浇水施肥 （做了，但方式不对）”

  4. 4、语音歧义：口语中因发音相同而引发歧义。比如重音、多音

    1. 例1：他终于想起来了。当重音为“想”时，即想到事情；当重音为“起来“时，即起来这个动作
    2. 例2：到2019年年底，厂里还欠款3000万元。这里的“还”可以读作“huan欠款”或“hai欠款”

二、随笔练习 ​

例1：（2018深圳）下列各项中，没有歧义的一项是：

  1. A. 研究员到这里工作才一个月，好多人还不认识。
  2. B. 演出期间，不少人遗憾地在剧场门口等着退票。
  3. C. 他走了一个多钟头了。
  4. D. 我看见他高兴得跳了起来。
  1. A项，“好多人还不认识”可以理解为“研究员不认识其他人”，也可以理解为“其他人不认识研究员”，存在歧义，排除；
  2. B项，没有歧义，当选；
  3. C项，“走”可以理解为走路，也可理解为离开，还可以理解为去世，“他走了一个多钟头了”存在歧义，排除；
  4. D项，“高兴得跳了起来”的主语不能确定是“我”还是“他”，存在歧义，排除。
  5. 故正确答案为B。

例2：（2014上海）没有歧义的是__________。

  1. A. 谁都看不上
  2. B. 照片放大了一点
  3. C. 吃多少就吃多少
  4. D. 我明天坐飞机回家
  1. A项，既可以理解为“这个人看不上别人”，也可以理解为“大家都看不上这个人”，存在歧义，排除；
  2. B项，既可以理解为“原来照片小，现在放大之后的尺寸达到了要求”，也可以理解为“照片放得过大，不符合要求”，存在歧义，排除；
  3. C项，既可以将“多少”理解为一个名词，表示数量的大小，句子意思为“能吃多少就吃多少”，也可以把“多”理解为副词，修饰“少”，句子意思为“尽可能地少吃”，存在歧义，排除；
  4. D项，句意明确，没有歧义，当选。
  5. 故正确答案为D。

例3：（2018上海执法）下列各项中，没有歧义的一项是：

  1. A. 研究员到这里工作才一个月，好多人还不认识。
  2. B. 演出期间，不少人遗憾地在剧场门口等着退票。
  3. C. 他走了一个多钟头了。
  4. D. 我看见他高兴得跳了起来。
  1. A项，“好多人还不认识”可以理解为“研究员不认识其他人”，也可以理解为“其他人不认识研究员”，存在歧义，排除；
  2. B项，没有歧义，当选；
  3. C项，“走”可以理解为走路，也可理解为离开，还可以理解为去世，“他走了一个多钟头了”存在歧义，排除；
  4. D项，我看见/他高兴地跳了起来“他跳”；我看见他/高兴地跳了起来“我跳”，存在歧义，排除。
  5. 故正确答案为B。

例4：（2015河北选调）下列句子没有歧义、表述准确的一项是：

  1. A. 如果你能抽出宝贵时间，亲自参加我们的婚礼，那是我们求之不得的。
  2. B. 老人80岁的时候，还记得哥哥参加学生运动时对自己的评价：一个温情主义者。
  3. C. 我看见他很高兴，和他聊起了很快就要举行的国庆晚会的筹备情况。
  4. D. 入秋以来，北京和河北的部分地区接连下了两场雨。
  1. B项中“自己”，到底是“我”还是”哥哥“，指代不明，排除；
  2. C项中“我”高兴还是“他”高兴，表义不明，排除；
  3. D项中“北京和河北的部分地区”是指“北京全部，河北的部分地区”还是“北京的部分地区和河北的部分地区”，表义不明，排除。
  4. 故正确答案为A。

例5：（2015河北选调）下列句子中，没有歧义的一句是：

  1. A. 伊朗的“核武”问题，已引起美国总统奥巴马的密切关注。对此，美国有的国际问题专家则表示乐观
  2. B. “依我看，这个考点最需要引起重视。”张老师补充道
  3. C. 我在楼顶上栽的几盆花，长得枝繁叶茂
  4. D. 厂长再次强调：“这个面子还是要的。”
  1. A项“对此”的“此”指代不明，既可指“伊朗的核武问题”也可指“奥巴马的密切关注”，存在歧义；
  2. B项“考点”既可以指考查的知识点，也可指考试的地点，有歧义；
  3. D项“面子”既可理解为“物体的表面”，也可理解为“情面”，存在歧义；
  4. C项没有歧义。

例6：（2018上海事业单位）下列语句有歧义的是（ ）。

  1. A. 你到底是否认真听取了我的意见？
  2. B. 究竟谁允许你到处乱倒建筑垃圾的？
  3. C. 莫非你也认为这件事我彻底做错了？
  4. D. 难道我老得连你也认不出来了吗？
  1. A项、B项、C项表述均无歧义，排除；
  2. D项：第一种：难道我老得（糊涂了）/连你也认不出来了吗？（意思是：我认不出你）。第二种：难道我老得（变化太大）/连你/也认不出来了吗？（意思是：你认不出我）存在歧义；。
  3. 故正确答案为D。

例7：（2019内蒙古通辽招聘）下列没有歧义的句子是（ ）。

  1. A. 我看见李浩扶着一位老人走下车来，手里拿着一只皮包
  2. B. 李煜认为杜甫有浓厚的门第等级观念的观点是值得商榷的
  3. C. 我校举办的数学、物理两个短训班开设了农用数学、农业机电课程
  4. D. 括号里的话如果是注释整个句子的，这种括号叫句外括号；句外括号要放在句末的标点之后
  1. A项，“手里拿着一只皮包”的主语不明确，无法确定是李浩还是老人；
  2. B项，本句中无法确定是谁的“观点”，可能是李煜的，也可能是杜甫的；
  3. C项，两门课程与两个短训班对应关系不明确，可以理解为两个短训班都开设了这两门课程，也可以理解为数学班开设了农用数学课程、物理班开设了农业机电课程；
  4. D项，无歧义。
  5. 故正确答案为D。

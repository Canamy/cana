比重专题 ​

字数: 4524 字 时长: 17 分钟 VIP文档

设：24年部分A，增长率a；24年总量B，增长率b

  1. 1、比重： 即部分在整体中所占的比例。
  2. 2、比重特征： Ａ 占 Ｂ 的比重、 Ｂ 中 Ａ 的占比

一、现期比重 ​

  1. 1、题型：问题时间和材料一致，出现占、比重等
  2. 2、公式：比重 = 部分总体，部分=总体 × 比重，总体=部分比重
  3. 3、速算：截位直除

例：（2022四川）2020年末，全国共有群众文化机构43687个，比上年末减少386个。其中乡镇综合文化站32825个，减少705个。年末全国群众文化机构从业人员185076人，比上年末减少4992人。其中具有高级职称的人员7075人，具有中级职称人员17969人。

  1. 问：2020年，全国群众文化机构从业人员中，中高级职称从业人员占比约为：
  2. A.8%  B.14%
  3. C.27%  D.38%
  1. 根据题干“2020年······中······占比约为”，可判定本题为现期比重问题。
  2. 定位文字材料最后一段可得：2020年末，全国群众文化机构从业人员185076人，其中具有高级职称的人员7075人，具有中级职称人员17969人。
  3. 根据公式：比重 = ，分量为7075+17969，整体为185076。
  4. 选项差距大，截位处理后为（7+18）÷185=25÷185，首位商1，因此选择B选项。

例：（2011浙江）2009年度全国“农民工总量”为22978万人，比上年增加436万人。其中“外出农民工”14533万人，比上年增加492万人。在外出农民工中，“住户中外出农民工”11567万人，比上年增加385万人；“举家外出农民工”2966万人，比上年增加107万人。 从性别看，男性外出农民工占65.1%，女性占34.9%。从年龄看，外出农民工以青壮年为主。其中，16-25岁占41.6%，26-30岁占20%，31-40岁占22.3%，41-50岁占11.9%，50岁以上的农民工占4.2%。从婚姻状况看，已婚的外出农民工占56%，未婚的占41.5%，其他占2.5%。

  1. 问：2009年，全国“31-50岁”的外出农民工约为()。
  2. A.8000万人  B.5000万人
  3. C.3000万人  D.1000万人
  1. 全国“31-50岁”外出的农民工占比为22.3%+11.9%=34.2%，根据 部分=总体 × 比重，则该年龄段的外出农民工人数为14533×34.2% ≈ 15000×(30%+4%)= 15000×30% + 15000×4% = 4500+600=5100万人，最接近的是B项。

例：（2018北京）2015年1~11月，北京市20个文化创意产业功能区实现收入7019.8亿元，同比增长7.5%，高于全市文化创意产业收入平均增速1.2个百分点，占全市文化创意产业收入的比重为68.8%。其中文化科技融合示范功能区实现收入3772.5亿元，同比增长7.8%，占20个功能区总收入的53.7%;文化金融融合功能区实现收入436.5亿元，同比增长10%，占20个功能区总收入的6.2%。

  1. 问：2015年1~11月，文化科技融合示范功能区实现收入占全市文化创意产业收入的比重约比文化金融融合功能区高多少个百分点?( )
  2. A.69  B.48
  3. C.33  D.25
  1. 方法一：根据2015年1~11月，北京市20个文化创意产业功能区实现收入7019.8亿元…占全市文化创意产业收入的比重为68.8%，可得全市文化创意收入为：7019.8/68.8%≈10200亿元，则上述问题列式为3772.5/10200-436.5/10200=(3772.5-436.5)/10200=3336/10200，首位商3，因此选择C选项。
  2. 方法二：若A占B的比重为a%，B占C的比重为b%，可得A/C=A/BxB/C=a%×b%，文化科技融合示范功能区实现收入占全市文化创意产业收入的比重为：68.8%×53.7%，文化金融融合功能区实现收入占全市文化创意产业收入的比重为68.8%×6.2%，二者差值为：68.8%×(53.7%-6.2%)=68.8%×47.5%≈33%。

二、基期比重 ​

  1. 1、题型：问题时间在材料之前，出现占、比重等
  2. 2、公式： × ，A部分的现期量；B总体的现期量；a部分的增长率；b总体的增长率。
  3. 3、公式推导：
    1. 基期的部分值=部分现期部分增长率
    2. 基期的整体值=总体现期总体增长率
    3. 所以基期比重=部分现期部分增长率÷总体现期总体增长率 = 部分现期总体现期 × 总体增长率部分增长率 = ×
  4. 4、速算：
    1. （1）截位直除A/B，一般选项中会出现A/B的占比，再看(1+b)/(1+a)与1的关系(>，<，=)，结合选项选答案。
    2. （2）拆1法。
    3. （3）等比例缩放法。

例：（2018辽宁）2018年1～7月份，社会消费品零售总额210752亿元，同比增长9.3%。其中，限额以上单位消费品零售额81125亿元，同比增长7.3%。

  1. 问：2017年1～7月，限额以上单位消费品零售额占社会消费品零售总额比重约为（）
  2. A.33.4%  B.37.6%
  3. C.38.5%  D.39.2%
  1. 看到“占”，想到比重问题。问2017年，给2018年，基期时间，为基期比重问题。
  2. 定位材料找数据，A=81125、a=7.3%、B=210752、b=9.3%，代入公式：基期比重=A/B*（1+b）/（1+a）=81125/210752×（1+9.3%）/（1+7.3%）。
  3. 先计算81125/210752≈38.5%。
  4. 再计算（1+9.3%）/（1+7.3%）＞1。
  5. 故原式=38.5%乘一个大于1的数＞38.5%，故选D。

例：（2017吉林）2012-2015年，我国国内生产总值年均增长率为 7.3%，远高于世界同期 2.4%（世界银行数据）的平均水平。对世界经济增长的贡献率平均约为26% 。2015年被称为新常态元年，我国GDP占世界的比重为15.5% ，比2012年高4个百分点。2015年GDP相当于美国的63.4% ，比2012年提高11个百分点。我国依然是世界经济增长的最重要引擎。

  1. 问：2012年，我国国内生产总值占世界的比重为：
  2. A.11.5%  B.4%
  3. C.6%  D.10.5%
  1. 由题干“2012年……占……比重”可知本题为基期比重问题。
  2. 定位材料第“2015我国GDP占世界的比重为15.5%，比2012年高4个百分点”，则2012年我国国内生产总值占世界的比重为15.5%-4%=11.5%，答案选A。

例：（2018浙江金华事业单位）2016年，全年全国艺术表演团体共演出230.60万场，比上年增长9.4%，其中赴农村演出151.60万场，增长9.0%;国内观众11.81亿人次，增长23.3%，其中农村观众6.21亿人次，比上年增长6.2%;总收入311.23亿元，比上年增长20.8%，其中演出收入130.86亿元，增长39.3%。

  1. 问：在2015年全国艺术表演团体演出场次中，赴农村演出占比约为：
  2. A.64.77%  B.65.74%
  3. C.65.97%  D.66.91%
  1. 让求的是2015年全国艺术表演团体演出场次中，赴农村演出的占比，和材料中2016年相比是基期，赴农村演出的场次是部分量，全国艺术表演团体演出场次是整体量。
  2. 根据文字材料“2016年，全年全国艺术表演团体共演出230.60万场，比上年增长9.4%，其中赴农村演出151.60万场，增长9.0%”。
  3. 代入基期比重公式=A/B×(1+b)/(1+a)=151.60/230.60×(1+9.4%)/(1+9.0%)
  4. 为了简化计算可以利用拆“1”法，代入拆“1”法公式，2015年全国艺术表演团体演出场次中，赴农村演出占比=151.60/230.60×(1+9.4%)/(1+9.0%)≈65.8%×(1+0.4%)≈66%。
  5. 因此，选择C选项。

三、两期比重 ​

  1. 1、题型：...占...比重，比上年上升/下降+百分点。比如：今年 A 占 B 的比重比过去某年改变多少百分点 (%)?
  2. 2、公式：两期比重差＝现期比重-基期比重=AB -（AB×1+b1+a） =AB×a−b1+a （A为部分现期值，B为整体现期值，a%是部分增长率，b%是整体增长率） 一般情况下，两期比重之差＜ |a-b|
  3. 3、结论：
    1. （1）当部分增长率(a)＞大于整体增长率(b)，则比重差值为正，即现期比重＞基期比重，比重呈上升趋势；
    2. （2）当部分增长率(a)＜大于整体增长率(b)，则比重差值为负，即现期比重＜基期比重，比重呈下降趋势；
  4. 4、计算方法
    1. （1）先判定比重上升还是下降
    2. （2）计算两期比重差< 丨a-b丨
    3. （3）如果第二步答案无法确定，则带入公式计算
  5. 小贴士：只要满足A÷B形式，并且求比上年上升/下降都可以使用该公式。

“一般情况下”，比重差小于增长率之差的理解

  1. 给出基期值，部分量为A，整体量为B，两者分别增长了a、b，则比重差=现期比重-基期比重=A(1+a)B(1+b)−AB=AB×a−b1+a
  2. A/B范围是0~1之间（通常小于1）
  3. 当a＞0，1+a＞1，所以11+a＜1
  4. 因此：A/B×11+a×（a-b）=(一个小于1的数)×(另一个小于1的数)×（a-b）
  5. 因此 比重差＜|a-b|
  6. 当a＜0，0＜1+a＜1，所以11+a＞1
  7. 一个小于1的数乘以一个大于1的数，结果可能大于1，也可能小于1。因此 比重差 与 |a-b| 的大小关系不确定。但是！ 由于现期比重(A/B)通常不会太小，而11+a也不会太大，在很多常见数据下，乘积A/B×11+a仍然可能小于1，所以比重差仍可能小于|a-b|
  8. 在 a＜0 时，要让 比重差 ＞ |a-b| 条件极其苛刻，需要现期比重非常接近1，同时 a 非常接近 -1。这在实际经济数据和考试中几乎不可能出现

例：（2016黑龙江）3月末，国有企业资产总额1054875.4亿元，同比增长12%；负债总额685766.3亿元，同比增长11.9%；所有者权益合计369109.1亿元，同比增长12.2%。其中，中央企业资产总额554658.3亿元，同比增长10.5%；负债总额363304亿元，同比增长10.4%；所有者权益为191354.4亿元，同比増长10.7%。

  1. 问：2015年3月末，中央企业所有者权益占国有企业总体比重比上年同期约：
  2. A.下降了0.7个百分点
  3. B.下降了1.5个百分点
  4. C.上升了0.7个百分点
  5. D.上升了1.5个百分点
  1. 根据题干中“2015年3月”“比上年同期”“占比重”，可以判定本题为两期比重问题。
  2. 先判断比重变化趋势；整体增长率(b)为12.2%，部分增长率（a）为10.7%，部分小于整体，则比重下降；排除C和D；
  3. 计算增长率差值丨10.7%-12.2%丨=1.5%，根据比重差小于增长率之差，需要选择小于1.5%的，只有A项满足。

例：（2017四川）2015年，纺织行业规模以上企业累计实现主营业务收入70713亿元，同比增长5.0%;实现利润总额3860亿元，同比增长5.4%;企业亏损面(亏损企业占所有企业比重)11.4%，比上年低0.1个百分点。

  1. 问：2015年，纺织行业规模以上企业主营业务利润率（利润总额/主营业务收入）比上年约：
  2. A.上升0.02个百分点
  3. B.上升0.4个百分点
  4. C.下降0.02个百分点
  5. D.下降0.4个百分点
  1. 由问题“2015年主营业务利润率（利润总额/主营业务收入）比上年约上升/下降几个百分点”，判断本题为两期比重问题。
  2. 定位材料，“2015，纺织行业规模以上企业累计实现主营业务收入70713亿元，同比增长5.0%；实现利润总额3860亿元，同比增长5.4%”。
  3. 根据两期比重差的结论，利润增长率(a)大于收入增长率(b)，故比重比上年应为上升，反之，比重比上年下降。代入数据a=5.4%>b=5.0%，利润率上升，排除C、D选项。
  4. 并且上升的幅度小于丨5.4%-5.0%丨=0.4%，即小于0.4个百分点，排除B选项。
  5. 因此，选择A选项。

例：（2019国家）2017年，A省全年完成快递业务量100.51亿件，同比增长31.0%。其中，同城快递业务量增长29.3%，异地快递业务量增长33.0%，国际和港澳台地区快递业务量增长33.1%。

  1. 问：2017年A省快递业务中，业务量占总业务量比重高于上年水平的分类是：
  2. A.仅国际和港澳台地区快递
  3. B.异地快递、国际和港澳台地区快递
  4. C.仅同城快递
  5. D.同城快递、异地快递
  1. 根据“2017年A省快递业务中，······占······比重高于上年水平的分类是···”，可判断此题为两期比重问题。
  2. 根据两期比重比较技巧可知，若部分增长率大于整体增长率，则比重上升即高于上年水平。
  3. 整体增长率总业务量增长率（b）为31.0%，其中部分增长率大于整体增长率（31.0%）的有异地快递（33.0%）和国际和港澳台快递（33.1%）。
  4. 因此，选择 B 选项。

例：（2017新疆）2015年国家自然科学基金委全年共接收173017项各类申请，同比增长约10%，择优资助各类项目40668项，比上年增加1579项。

  1. 问：2015年国家自然科学基金委全年平均资助率较2014年约：
  2. A.增加1.4%
  3. B.减少1.4%
  4. C.增加2.1%
  5. D.减少2.1%
  1. 根据题干所求“2015年国家自然科学基金委全年平均资助率较2014年”和材料第二段“平均资助率（资助项目占接收申请项目的比重）”可判定此题为两期比重问题。
  2. 可求得2015年择优资助各类项目同比增长率为 1579÷（40668-1579）≈1580÷39100≈4%。
  3. 首先为负值，根据两期比重公式：自然科学基金委增长率(b)为10%，择优资助各类项目增长率（a）为4%，部分小于整体，则比重下降；排除A和C；
  4. 根据比重差小于增长率之差判断不出，只能列式计算。
  5. 40668173017×4%−10%1+4%
  6. 40668÷173017＜1/4=25%，6%÷104%＜6%，则比重差＜25%×6%=1.5%，排除D选项。故正确答案为B。

四、拓展：比重增幅 ​

  1. 1、题型：求比重（或占比）的增幅（或增长量）。
  2. 2、方法：现期比重 - 基期比重（本质就是两期比重差）
  3. 3、分析：很多局长看到增幅就是求增长率(现期比重 - 基期比重)÷基期比重，其实是错误的；比重的增幅不能用增长率公式计算，因为比重本身是一个比例值，以百分数形式呈现。如果使用百分比变化来计算增幅，容易造成双重百分比的混淆。
    1. 例如：基期比重为40%，现期比重为50%。
    2. 差值（绝对增幅）：50% - 40% = 10%。
    3. 百分比变化（相对增幅）：(50% - 40%) / 40% × 100% = 25%。
    4. 这里，25%的“增幅”可能被误解为“比重本身增加了25%”，但实际是比重值从40%增加到50%，绝对增加量是10个百分点。使用差值（10个百分点）更清晰地表示实际增加的量，而百分比变化（25%）容易引起误解，因为它依赖于基期值（40%），且可能被错误解读为比例的内部变化。

例：（2022国考）

  1. 问：已知2020年中国IC封装市场规模同比增长x亿元，IC封装市场中IC先进封装市场规模占比同比增长y个百分点，而2020年往后中国IC封装市场规模及IC先进封装市场规模占比每年都分别同比增长x亿元和y个百分点，则到“十四五”最后一年（2025年），中国IC先进封装市场规模将达到多少亿元?
  2. A.433  B.469
  3. C.521  D.575
  1. 根据题干“······‘十四五’最后一年（2025年）······将达到多少亿元”，可判定本题为现期计算问题。题目存在x(增长量)，y未知数(占比同比增长...百分点，求比重增幅)，先求未知数。
  2. 定位统计图可得：2020年、2019年IC封装市场规模分别为2478.9亿元、2349.7亿元，则x=2478.9-2349.7=129.2亿元；
  3. IC先进封装市场规模占比分别为13.26%、12.58%。则y=13.26%-12.58%=0.68%，即0.68个百分点。
  4. 问2025年，题干是2020年，x和y不变，年份差为5。根据公式：现期量=基期量+增长量，则2025年IC封装市场规模=2478.9+129.2×5=3124.9亿元，IC先进封装市场规模占比=13.26%+0.68%×5=16.66%。
  5. 根据公式：部分=总体×比重，则2025年中国IC先进封装市场规模将达到3124.9×16.66%≈3125×16≈521亿元。
  6. 故正确答案为C。
